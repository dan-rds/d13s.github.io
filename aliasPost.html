ALIASES FOR NETWORK BOOTERS

Posted by Daniel Richards on August 8, 2017

I made a simple bash script that adds useful aliases if you are using dnsmasq and emacs. On any ubuntu machine, simply download or clone this repo:

https://github.com/danieldrichards/linuxAliases.git

After that, run the script. This will add th folowing aliases:

netin = sudo emacs /etc/network/interfaces
leases = tail -f /var/lib/misc/dnsmasq.leases
log = tail -f /var/log/syslog
renet = sudo service networking restart; sudo service networking status
hgrep = history | grep "$@"â€™
remasq = sudo service dnsmasq restart; service dnsmasq status

These will come in handy when configuring and debugging your DNS, DHCP, and TFTP servers.
